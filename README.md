# Low-Level Design (LLD) Revision

This repository contains preparation materials and code implementations for Low-Level Design (LLD) interviews, specifically targeted towards a Senior Backend Engineer (SDE2) level.

The core roadmap and study plan are outlined in the [initial-plan.md](initial-plan.md) file, which serves as the primary guide for this revision.

## Repository Structure

- **`design-patterns/`**: Contains comprehensive notes (`DESIGN_PATTERNS_REVISION.md`) and real-world Java code implementations for Creational, Structural, and Behavioral GoF design patterns.
- **`java-core/`**: Focuses on core Java concepts crucial for LLD, including multithreading, parallel task execution, and custom data structure implementations (like HashMaps).
- **`lld-practice-1-2024/`**: A collection of older (2024) LLD practice problems, such as ATM, Elevator System, Payment Gateway, Splitwise, and more.
- **`lld-practice-2-2026/`**: Current (2026) implementations of high-frequency LLD problems (e.g., Parking Lot) designed with senior-level constraints, focusing on concurrency, thread safety, and extensibility.
- **`solid-principles/`**: Contains notes (`SOLID_REVISION.md`) and source code examples demonstrating the application of SOLID object-oriented design principles.
